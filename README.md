# WAM-TOOLS
A set of tools that most people usually use
## First release V 1.0
### Tools List
- Password generator
- QR code generator
- CSS Box shadow generator
- MD5 hash generator
- Base 64 encode
- Base 64 decode
- Convert RGB color To Hex color
- Convert Hex color To RGB color
- Text tools
    - Make text uppercase
    - Make text lowercase
    - Uppercase the first letter of the text
    - Number of letters
    - Number of words
